# **App Name**: Ahlan Arabic

## Core Features:

- User Authentication: Secure role-based authentication (teachers, kids) with session management.
- Interactive Sessions: Teachers create phrase-based sessions, kids join via PIN. Supports Tunisian dialect, MSA, French, and English.
- Real-time Feedback: Kids submit answers, teachers review/correct in real-time, and assign scores based on accuracy.
- Score Generation: The system decides when to include an incentive based on past student progress by incorporating a tool that promotes retention using techniques found on previously validated scientific papers. It computes points per answer and total session score and adjusts them dynamically based on historical trends and comparison between expected results and other aggregated datasets. It rewards and incentivizes performance increases and reaching benchmarks.
- Gamified Leaderboard: Real-time leaderboard display with badges and persistent progress tracking to motivate kids.
- Multilingual Interface: Interface and instructions available in Arabic, French, and English, with easy toggle option.
- Admin Dashboard: Teacher dashboard for session control, analytics, history logs, and secure access control.

## Style Guidelines:

- Primary color: Vibrant orange (#FF8C00) to create a playful and inviting atmosphere.
- Background color: Light orange (#FFF0D4), providing a soft and unobtrusive backdrop.
- Accent color: Analogous yellow (#FFD700), which creates visual interest.
- Body and headline font: 'PT Sans' for a friendly and readable appearance. Its modern but approachable style is suited to children's education.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use playful and culturally relevant icons to represent phrases and actions.
- Subtle animations for correct answers, score updates, and leaderboard changes.
